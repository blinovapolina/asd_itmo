## Задание

«проверка существования ключа».

| Input | Output |
| ----- | ------ |
| 8<br/>A 2<br/>A 5<br/>A 3<br/>? 2<br/>? 4<br/>A 2<br/>D 2<br/>? 2| Y<br/>N<br/>N|
